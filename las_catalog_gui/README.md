# GUI-каталогизатор LAS-файлов на основе Tkinter

## Описание

Приложение с графическим интерфейсом для каталогизации LAS-файлов. Позволяет:
- Выбрать каталог через диалоговое окно
- Рекурсивно найти все *.LAS файлы в подкаталогах
- Отобразить информацию о скважинах и кривых в таблице
- Экспортировать данные в TSV-файл (UTF-8, разделитель - табуляция)

## Требования

- Python 3.x
- tkinter (обычно входит в стандартную поставку Python)
- las_memory.py (должен находиться в родительской директории)

## Запуск

```bash
python las_catalog_gui.py
```

## Функционал

1. **Выбор каталога**: Кнопка "Выбрать каталог" открывает диалог для выбора директории с LAS-файлами
2. **Автоматическое сканирование**: После выбора каталога происходит рекурсивный поиск всех .LAS файлов
3. **Отображение данных**: Информация отображается в таблице со следующими колонками:
   - Путь к файлу (с разделителями \)
   - WELL (название скважины)
   - FLD (поле)
   - STRT (начальная глубина)
   - STOP (конечная глубина)
   - STEP (шаг)
   - MNEM (имя кривой)
   - UNIT (единицы измерения)
   - DESCR (описание кривой)
4. **Экспорт**: Кнопка "Экспорт в TSV" позволяет сохранить результаты в файл

## Зависимости

Модуль использует `las_memory.LasMemory` для парсинга LAS-файлов.
