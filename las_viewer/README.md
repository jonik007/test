# LAS File Viewer

Flask-приложение для просмотра LAS файлов с возможностью перетаскивания (drag-and-drop). Приложение отображает метаинформацию скважины и таблицу с данными кривых.

## Возможности

- 📁 **Drag-and-Drop интерфейс** - просто перетащите LAS файл в браузер
- 📋 **Метаинформация** - отображение всех основных параметров скважины
- 📈 **Таблица данных** - просмотр данных кривых в табличном формате
- 🔤 **Автодетект кодировки** - поддержка русских кодировок (Windows-1251, DOS 866, UTF-8)
- ⚡ **Работа из памяти** - файлы не сохраняются на диск сервера
- 📊 **Ограничение вывода** - показываются первые 100 строк для производительности

## Установка

1. Убедитесь, что у вас установлен Python 3.7+

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Убедитесь, что библиотека `las_memory` доступна (должна находиться в соседней директории или быть установлена)

## Запуск

```bash
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000

## Использование

1. Откройте приложение в браузере
2. Перетащите LAS файл (.las) в область загрузки или кликните для выбора файла
3. Дождитесь обработки файла
4. Просмотрите метаинформацию скважины и данные кривых

## API

### POST /upload

Загрузка LAS файла для обработки.

**Параметры:**
- `file` (multipart/form-data): LAS файл для обработки

**Ответ:**
```json
{
  "success": true,
  "metadata": {
    "well_name": "Название скважины",
    "field": "Месторождение",
    ...
  },
  "curves": [...],
  "table_headers": ["DEPT", "GR", "RHOB", ...],
  "table_data": [...],
  "total_rows": 1000,
  "displayed_rows": 100
}
```

## Особенности

- Поддержка различных версий LAS файлов (1.2, 2.0)
- Автоматическое определение кодировки для корректного отображения кириллицы
- Обработка специальных значений (null values)
- Адаптивный дизайн для мобильных устройств

## Требования

- Python 3.7+
- Flask 2.3+
- NumPy 1.20+
- Библиотека `las_memory` (локальная версия)
